1. 星期一：配合醫師門診時間，下午(13:00-17:00)
2. 星期二：遠端，可以選擇(8:00-12:00 或 13:00-17:00）
3. 星期三：配合醫師門診時間，早上(8:00-12:00)
4. 星期四：遠端，可以選擇(8:00-12:00 或 13:00-17:00）
5. 星期五：配合醫師門診時間，早上(8:00-12:00)
